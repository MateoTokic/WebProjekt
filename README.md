# **WebProject**

A website for booking vacations in Croatia, designed with **PHP**, **HTML**, **CSS**, and **JavaScript**. The platform allows administrators to add vacation offers while registered users can book vacations, read and write reviews, and explore information about Croatia.

---

## **Features**

- **Admin Panel**: Add and manage vacation offers.
- **User Functionality**: Book vacations, write reviews, and explore tourist information about Croatia.
- **Interactive UI**: A user-friendly interface for easy navigation and interaction.

---

## **Technologies Used**

- **PHP**: Backend logic and server-side processing.
- **HTML & CSS**: Structuring and styling of web pages.
- **JavaScript**: Interactivity and dynamic user experiences.
